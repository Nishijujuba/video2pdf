# Legacy Workspace Maintenance Context

Status: supporting.

This context owns evidence-based classification and relocation of historical video documentation directories. Video Workflow owns new-run naming, final PDF naming, Run Migration, and every lifecycle rule for active runs.

## Historical Directory Identity Evidence

Durable material inside a historical directory that identifies the delivered video or article and supports a trustworthy artifact date. Filesystem creation time alone has no identity authority.

## Valid Video Output Directory

A historical video documentation directory eligible for migration because its internal durable artifacts establish a final-delivery identity. Scratch files, logs, caches, partial downloads, and inferred dates alone cannot establish validity.

## Video Artifact Date

The earliest trustworthy date derived from durable artifacts for classifying and naming historical video documentation. It belongs to relocation evidence and cannot define a new Video Workflow Run start time.

## High Confidence Video Migration

A historical relocation classification whose internal evidence supports deterministic destination identity and whole-directory movement without an identity guess.

## Low Confidence Video Migration

A historical relocation classification used when identity evidence is missing or contradictory, or when a destination conflict cannot be resolved mechanically. It preserves the directory for explicit human review.

## Historical Workspace Relocation

The movement of one classified historical video documentation directory as a complete unit into its resolved workspace location. It preserves the directory's internal relative layout and evidence relationships.
