# YouTube Source Judgment Policy

Use only the YouTube candidates authorized by the Task Envelope and Decision Skeleton. Prefer a complete usable English subtitle when the policy classifies the source as language learning. Apply the exact language priority supplied by the Skeleton for other content.

Prefer a usable manual track over an automatic track at the same language priority. Select Whisper only when the authorized subtitle candidates are absent or technically unusable and the Skeleton permits Whisper with an audio candidate. Record partial coverage, automatic-track quality limits, or unavailable fallback as explicit known gaps.

Cookie handling, authentication classification, JavaScript runtime selection, platform commands, downloader options, canonical naming, media probes, and fingerprints belong to the YouTube Platform Adapter and Kernel providers. The judgment must never contain Cookie material, Cookie paths, raw command output, or authentication diagnostics.
